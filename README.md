# 🎆 Pyrotechnic

**A browser-based graphic design canvas — the spiritual successor to Adobe Fireworks.**

Create OGP images, banners, and thumbnails directly in your browser. Export to PNG, JPEG, or SVG.  
Place text, shapes, and lines freely, and automate your designs with **JSON macros** — generated by AI or written by hand.

---

## ✨ Features

### 🖌️ Visual Editing
- Place rectangles, ellipses, lines, and text with drag-and-drop
- Edit color, font, opacity, corner radius, drop shadow, and blend mode in the Properties panel
- Layer management — reorder, show/hide, lock, and flatten layers
- Multi-select, group move, and transform handles (scale, rotate)
- Pen tool for freeform paths; brush and eraser for bitmap painting

### 📐 Canvas
- Any canvas size (OGP 1200×630, YouTube 1280×720, Instagram 1080×1080, etc.)
- Rulers, grid snap, and smart guides
- Zoom and pan; keyboard shortcuts for common operations
- Undo / Redo history

### 📦 Export
- **Slice tool** — draw export regions directly on the canvas
- Export individual slices or all slices at once as PNG, JPEG, or SVG

### 🎬 Animation (Multi-State)
- Add multiple frames (states) to create frame-by-frame animations
- Play / pause preview in the States panel with per-frame delay control
- Built-in **Bouncing Ball** animation sample demonstrating Squash & Stretch

### 🤖 JSON Macro System
Describe canvas operations as JSON and run them in one click.  
**Ask ChatGPT, Claude, Gemini, or any AI to generate macros for you.**

```json
{
  "schema": "1.0",
  "title": "OGP Image",
  "commands": [
    { "command": "set_canvas", "width": 1200, "height": 630 },
    { "command": "clear_canvas" },
    { "command": "add_rect", "x": 0, "y": 0, "width": 1200, "height": 630, "fill": "#0f172a" },
    { "command": "add_text", "x": 80, "y": 240, "width": 1040, "height": 120,
      "text": "Your Headline Here", "fontSize": 68, "fontFamily": "Outfit",
      "fontWeight": "bold", "fill": "#f8fafc", "textAlign": "center" }
  ]
}
```

- **Run** — paste any macro JSON and execute it instantly
- **Export** — convert the current canvas back to a macro JSON file
- **Samples** — load the OGP template or Bouncing Ball animation with one click

## 🤖 Generating Macros with AI

Copy the contents of [`PYROTECHNIC_MACRO_PROMPT.md`](./PYROTECHNIC_MACRO_PROMPT.md) and paste it as a **system prompt** into ChatGPT, Claude, Gemini, or any other AI.

The AI will generate valid Pyrotechnic macro JSON from plain-language design requests.

### Workflow

1. Paste `PYROTECHNIC_MACRO_PROMPT.md` into your AI as a system prompt
2. Describe what you want: *"A dark navy OGP image with a gold accent line and bold headline"*
3. The AI outputs a JSON macro
4. Paste it into the **Macro panel** and click **Run**

### Example prompts

> *"Create an OGP image (1200×630) with a deep navy background, gold accent lines, and the headline 'Design Smarter, Not Harder' centered in white."*

> *"Make a vivid red-and-black YouTube thumbnail (1280×720) with the title 'How I Built This' in large bold text on the left."*

> *"A minimal white Instagram post (1080×1080) with the quote 'Less is More' and my username at the bottom."*

## 📋 Macro Command Reference

| Command | Description |
|---|---|
| `set_canvas` | Set canvas width and height |
| `clear_canvas` | Remove all objects from the canvas |
| `add_rect` | Add a rectangle (supports corner radius, shadow, blend mode) |
| `add_ellipse` | Add an ellipse or circle |
| `add_line` | Add a straight line |
| `add_text` | Add a text element (font, weight, style, alignment, shadow) |
| `add_slice` | Add an export region (PNG, JPEG, or SVG) |
| `add_state` | Add an animation frame |
| `clear_all_states` | Remove all animation frames |

Full schema and field reference: [`PYROTECHNIC_MACRO_PROMPT.md`](./PYROTECHNIC_MACRO_PROMPT.md)
